一个项目总文件夹就是一个仓库

我的习惯是Github或Gitee创建一个与本地项目总文件夹名字一致的仓库

我喜欢在github和gitee的用户名和个人邮箱都是同一个,看着舒服…

连通本地和远程存储库的一个重要操作是ssh连接,建立了这个连接以后后续很多操作都会变得便捷，包括github pages也需要它

再安装好了git后，设置git 用户名和密码以及联系邮箱
